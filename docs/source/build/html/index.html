Welcome to hexagonal’s documentation!¶

hexagonal is an LMS (library management system) build as a first year bachelors’ project at Innopolis University. Class diagram can be found at hexagonal.docs.hexagonal-classes.uml. Also a rendered version here.

A good starting point would be docs for all models in hexagonal.model.

It should also be noted that as a dynamic by nature language, Python doesn’t have native support for class attribute documentation. This is also a bad practice, because it hurts readability of code.
