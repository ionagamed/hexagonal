hexagonal.auth.helpers.minimum_access_level(level)¶

Decorator for bound functions which restricts the allowed users to at least level access level. Throws hexagonal.auth.jsonrpc.exceptions.JSONRPCAccessDeniedException when user isn’t allowed.
