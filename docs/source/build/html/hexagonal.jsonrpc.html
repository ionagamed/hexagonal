hexagonal.jsonrpc.call(name, args=None, optional=None)¶

Perform the actual call of a function by name and args object.

args can be None - then function is called without arguments. args can be a list - then function is called with positional arguments. args can be a dict - then function is called with named arguments.

optional is a dict with named arguments which are passed to a function, if present in the declaration.

Parameters:
  • name – function name
  • args – arguments of the function
  • optional – optional arguments of the function
Returns:

result of function call
